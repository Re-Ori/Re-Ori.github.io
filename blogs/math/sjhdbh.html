三角恒等变换

 

sin⁡(α+β)=sin⁡αcos⁡β+cos⁡αsin⁡β

cos⁡(α+β)=cos⁡αcos⁡β−sin⁡αsin⁡β

tan⁡(α+β)=tan⁡α+tan⁡β1−tanαtan⁡β (sin⁡xcos⁡x)

0 辅助角公式

asin⁡α+bcos⁡α=a2+b2sin⁡(α+φ)，其中 φ 的终边经过(a,b)

1 倍角公式

sin⁡2x=2sin⁡xcos⁡x

cos⁡2x=cos2⁡x−sin2⁡x=2cos2⁡x−1=1−2sin2⁡x

tan⁡2x=2tan⁡x1−tan2⁡x

2 升幂公式

1±sin⁡2x=(sin⁡x±cos⁡x)2 (1-1)

1+cos⁡2x=2cos2x (1-2[3])

1−cos⁡2x=2sin2x (1-2[2])

3 降幂公式

sin2⁡α=1−cos⁡2α2 (1-1)

cos2⁡α=1+cos⁡2α2 (2-2)

tan2⁡α=1−cos⁡2α1+cos⁡2α (2-3)

4 万能公式

sin⁡2x=2tan⁡x1+tan2⁡x (1−1sin2α+cos2α)

cos⁡2x=1−tan2x1+tan2x (1−2[1]sin2α+cos2α)

tan⁡2x=2tan⁡x1−tan2⁡x (1-3)

5 半角公式

sin⁡x2=±1−cos⁡x2 (3-3)

cos⁡x2=±1+cos⁡x2 (3-2)

tan⁡x2=±1−cos⁡x1+cos⁡x (4−14−2)

Tip
符号由sin⁡/cos⁡/tan⁡α2的正负决定

6 积化和差公式

cos⁡acos⁡b=12[cos⁡(a+b)+cos⁡(a−b)] ( cos⁡(a+b)+cos⁡(a−b)=2cos⁡acos⁡b )

sin⁡asin⁡b=12[cos⁡(a−b)−cos⁡(a+b)] ( cos⁡(a−b)−cos⁡(a+b)=2sin⁡asin⁡b )

sin⁡acos⁡b=12[sin⁡(a+b)+sin⁡(a−b)] ( sin⁡(a+b)+sin⁡(a−b)=2sin⁡acos⁡b )

cos⁡asin⁡b=12[sin⁡(a+b)−sin⁡(a−b)] ( sin⁡(a+b)−sin⁡(a−b)=2cos⁡asin⁡b )

7 和差化积公式

cos⁡x+cos⁡y=2cos⁡x+y2cos⁡x−y2 ( 2cos⁡x+y2cos⁡x−y2=cos⁡(x+y2+x−y2)+cos⁡(x+y2−x−y2)=cos⁡x+cos⁡y )

cos⁡x−cos⁡y=−2sin⁡x+y2sin⁡x−y2 ( −2sin⁡x+y2sin⁡x−y2=cos⁡(x+y2+x−y2)−cos⁡(x+y2−x−y2)=cos⁡x−cos⁡y )

sin⁡x+sin⁡y=2sin⁡x+y2cos⁡x−y2 ( 2sin⁡x+y2cos⁡x−y2=sin⁡(x+y2+x−y2)+sin⁡(x+y2−x−y2)=sin⁡x+sin⁡y )

sin⁡x−sin⁡y=2cos⁡x+y2sin⁡x−y2 ( 2cos⁡x+y2sin⁡x−y2=sin⁡(x+y2+x−y2)−sin⁡(x+y2−x−y2)=sin⁡x−sin⁡y )
