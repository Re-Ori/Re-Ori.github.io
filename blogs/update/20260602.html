🔒 SSL 证书支持

服务器现在可以顺利完成 Let's Encrypt 的 HTTP-01 域名验证，申请 SSL 证书。

以后访问本站就是 HTTPS 而不是 HTTP 了。

🚀 Giscus 评论区国内加速

Giscus 评论区现在会优先使用本地代理：

  • Giscus 客户端 JS 从本站服务器加载（绕过 Vercel CDN）
  • GitHub GraphQL API 请求通过本站服务器代理转发
  • 无代理环境（如 GitHub Pages）自动降级到 CDN 直连，评论区不中断

🌐 P2P 直连空间

新增 P2P 点对点通信功能，所有数据传输不走服务器：

  • 文字聊天 — 通过 WebRTC DataChannel 实时传输
  • 文件传输 — 支持任意文件类型，分块传输
  • 图片分享 — 拖拽或选择图片直接发送，接收方即时预览
  • 屏幕共享 — 基于 WebRTC 媒体流，一键分享屏幕
  • 房间机制 — 创建/加入房间，分享链接即可邀请他人

入口：导航栏「P2P」按钮 / 首页「🌐 P2P 直连空间」卡片

🧠 智能后端检测

P2P 和 Giscus 代理都带有后端可用性检测：

  • 在 reori.cn 上 → 自动启用 P2P 和 Giscus 代理
  • 在 GitHub Pages 等纯静态环境 → 自动检测并降级，功能不受影响
