1.5.0 20260611 更新

1. 新增 Origin BBS 论坛系统,支持自由创建帖子、发布信息,Markdown 格式,可回复互动,全屏编辑器。内置账号系统运行于服务器端,账号通过 JSON 文件手动管理（暂不支持注册）,支持自定义标签和角色权限。点击用户名进入个人主页查看其帖子列表,身份标识系统以不同颜色区分管理员、自己、贴主及自定义标签

2. 新增短链生成器,支持将长链接转为短链接。匿名用户可直接使用（72 小时有效期）,登录 BBS 账号后永久有效。页面增加环境检测,自动识别服务器可用性

3. P2P 直连空间与短链生成器整合至工具页,通过导航栏和首页均可访问

4. 导航栏支持自动高亮当前页面,可通过 JSON 配置指定高亮项或手动传入参数覆盖
